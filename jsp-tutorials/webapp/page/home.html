Easy JSP

Easy JSP သည် Myanmar Java User ဘလောဂ်တွင်ရေးသားထားသော JSP Tutorials များ၏ နမှုနာများကို စုစည်းထားသော နမှုနာ စာမျက်နှာများဖြစ်ကြပါသည်။

Tutorials မှ အခန်းအသီးသီး၏ စာမျက်နှာများကို သွားရောက်နိုင်ပြီး အလုပ်လုပ်ပုံကို လက်တွေ့စမ်းသပ်ကြည့်နိုင်ပါသည်။ ဤပရိုဂျက်သည် Web Module အား Cloudbees ၏ Run Cloud အပေါ်တွင် တည်ဆောက်ထားပြီး၊ အသုံးပြုထားသော Source Code များမှာမူ Google Code တွင် သိမ်းဆည်းထားပါသည်။ ထို့အတွက် Source Code များကို လေ့လာလိုပါက Sources မှ တဆင့်သွားရောက် ကြည့်ရှုနိုင်ပါသည်။ ရှင်းလင်းချက်များကို Myanmar Java User ဘလောဂ်တွင် လေ့လာနိုင်ပါသည်။

ဤပရိုဂျက် ဖြစ်မြောက်ရေးအတွက် အခမဲ့အသုံးပြုခဲ့သော Blogger၊ Cloudbees၊ Google Code နှင့် jquery အား ကျေးဇူးတင်ရှိကြောင်း ဤနေရာမှပြောကြားလိုပါသည်။
