  • ကားအသစ်သွင်းရန်

ရှင်းလင်းချက်

ယခင်တစ်ခေါက် Episode5 ၏ နမှုနာကို MVC အား အခြေခံ၍ ပြုပြင်ရေးသားထားပါသည်။

  1. Controller နေရာတွင် အသုံးပြုသော Servlet အတွင်းမှ Business ID အား ကြည့်ကာ အသုံးပြုလိုသည့် Model အားခေါ်ယူ အလုပ်လုပ်စေပြီး၊ Flows တွင် သတ်မှတ်ထားသော View ဖြင့် ဖော်ပြနိုင်ရန် ပြုပြင်ထားပါသည်။
  2. လုပ်ဆောင်ချက်တစ်ခု တိုးတိုင်း လိုအပ်သည့် Configuration များအားပြုပြင်ပြီး၊ Model နှင့် View အား ရေးသားရုံသာပင် ဖြစ်၏။ ယခင်ရှိပြီး အပလီများအား ပြုပြင်ရန် မလိုအပ်ပေ။
  3. ဤနေတစ်ခေါက်တွင် အသေးစိတ်ဖော်ပြမည် မဟုတ်သော်လည်း MySQL အား အသုံးပြုထားပါသည်။
