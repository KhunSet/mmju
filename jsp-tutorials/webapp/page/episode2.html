ရှင်းလင်းချက်

JSP Action ကို အသုံးပြုထားသော့် နမှုနာများဖြစ်ပါသည်။

ဤနမှုနာဖြင့် JSP စာမျက်နှာ တစ်ခုကို ရေးသားရန် အခြေခံလိုအပ်ချက်များကို လေ့လာနိုင်ပါသည်။ JSP စာမျက်နှာတွင် အသုံးပြုနိုင်သော အခြေခံ Tag များဖြစ်သော Script နှင့် အသုံးပြုနိုင်သော Object များကို လေ့လာမှတ်သားနိုင်ပါသည်။

ဤနမှုနာထဲတွင် jsp:useBean, jsp:setProperty, jsp:getProperty, jsp:forward နှင့် jsp: include အစရှိသည့် jsp action များကို အသုံးပြုထားပါသည်။
